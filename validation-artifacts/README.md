# Validation Artifacts

This folder contains the visual proof of successful architecture deployment, including:
1.  **Salesforce SSO Success:** Verification of successful SAML login.
2.  **Okta Group Rules:** The IF/THEN automation logic.
3.  **Automated Provisioning:** User profiles showing applications assigned by group rules.
4.  **Leaver Flow Deactivation:** Proof of access revocation upon AD account disablement.
